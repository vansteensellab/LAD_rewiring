LAD rewiring data: CRISPRa in RPE cells – linking DamID changes to mRNA-seq tracks.

Bas van Steensel, Nov 2019.

Introduction

This script plots gene expression related figure panels of Brueckner et al.

Description of Data

  • DamID data preprocessed as in scripts LAD_Rewiring_RPE_CRISPRa_[.]*_BvS[0-9]{6}.Rhtml. Data are average smoothed DamID scores per gene.
  • Genome annotation is Ensembl build 75. This build is a bit old, but it is what the NKI Genomics Core Facility uses for RNAseq data processing.
  • Expression data for 8 targeted genes by Laura Brueckner. Two replicates each.
  • mRNA-seq data re-processed by Tom van Schaik.
  • This script requires a new version of the LAD rewiring functions set: LAD_Rewiring_functions_BvS191112.R or later.

Gene annotation data

Load Refseq annotations and convert to bed format dataframe. The latter is needed for the plotting of gene tracks using the Sushi package.

#Ensembl hg19 build 75 downloaded from    feb2014.archive.ensembl.org/biomart/martview/4c34fe2f4e47c6b55a212cc95e3c952b
#load:
Ensembl.75<-read.delim("./data/ensEMBL/hg19/ensEMBL75_GRCh37.p13.txt", head=TRUE, sep="\t", as.is=TRUE)
#select and rename columns to make a .bed like dataframe:
Ensembl.75<-Ensembl.75[,c(3,4,5,7,6,6,1)]
names(Ensembl.75)<-c("chrom", "start", "stop", "gene", "score","strand","ensembl.gene.id")
#only on full chromosomes:
Ensembl.75<-Ensembl.75[Ensembl.75$chrom %in% c(1:22,"X","Y"),]
#reformat some stuff:
Ensembl.75$chrom<-paste0("chr",Ensembl.75$chrom)
Ensembl.75$strand<-ifelse(Ensembl.75$strand==-1, "-", "+")
#remove redundancies
Ensembl.75<-unique(Ensembl.75)
#remove questionable genes (mostly absent in UCSC):
Ensembl.75<-Ensembl.75[-grep("^RP11", Ensembl.75$gene, value=FALSE),]
Ensembl.75<-Ensembl.75[-grep("^AC[0-9]{6}[.][0-9]*", Ensembl.75$gene, value=FALSE),]
Ensembl.75<-Ensembl.75[Ensembl.75$stop - Ensembl.75$start > 200,] #remove very small genes, as the plotGenes() function of Sushi cannot cope with them
#how many genes in the end:
dim(Ensembl.75)

#as genomic ranges object:
GeneGR<-makeGRangesFromDataFrame(Ensembl.75)
